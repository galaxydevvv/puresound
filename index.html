Configure Drive Access

This player reads your shared Drive folder using the Google Drive API. Copy config.example.js to config.js and add your key there for direct browser access.

  1. Create an API key in Google Cloud Console.
  2. Enable the Google Drive API for that project.
  3. Paste the key into config.js and lock it to your site referrer.

Any key shipped to the browser can still be inspected. If you want real secrecy, move Drive requests behind a backend proxy.
